SaveySaver

SaveySaver Project for UX Design

Project Overview SavvySaver is a budgeting app designed with young adults in mind, empowering them with the tools and guidance they need to manage their finances effectively. From creating personalized budgets to tracking expenses and setting financial goals, SavvySaver provides a comprehensive solution to help users achieve financial stability and success.

Purpose and Goals This project serves multiple purposes:

  1. Practical Experience By working on SavvySaver, I am gaining hands-on experience in UX design, and UI design. This project allows me to apply theoretical knowledge in a real-world context, honing my skills and preparing me for future professional opportunities.
  2. Academic Requirement SavvySaver is a key component of my Google UX Design class on Coursera. The project enables me to fulfill course requirements while demonstrating my ability to create user-centered design solutions.
  3. Financial Literacy SavvySaver aims to address a critical need for financial literacy among young adults. By providing intuitive budgeting tools and personalized financial guidance, the app helps users take control of their financial future and achieve their financial milestones.

Project Features

  • Personalized Budgets Users can create customized budgets based on their income, expenses, and financial goals.
  • Expense Tracking The app allows users to track their spending and identify areas for improvement.
  • Goal Setting SavvySaver helps users set and achieve financial goals, whether it's saving for a home, a car, or greater financial stability.
  • Financial Tips The app provides practical tips and advice to help users save money and make informed financial decisions.
